# CSIRO---Image2Biomass-Prediction
Modèle de prédiction de biomasse végétale pour optimiser le pâturage durable. Utilise deep learning sur images de pâturages australiens + données NDVI pour estimer 5 types de biomasse (herbe verte/morte, trèfle, totale). Impact direct sur welfare animal et santé des sols. Compétition CSIRO/MLA - 75k$ prize pool.
